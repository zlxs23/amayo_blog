怎样使用 Pelican

2016-09-14
By Amayou

Usage of Pelican

  1. 切换工作空间至 amayo_blog
  2. 在目录下 content 编写 markdown 文件 格式： Title: (文章名) Date: (文章书写时间) Category: (存档名) Tags: (标签名) Slug: (html名) Author: (作者) Summary: (简短介绍)
        书写文章 正文
    1. 保存 markdown 至 content 文件夹下
    2. 启动（激活）active amayo py 环境
    3. 切换至 amayo_blog 输入 pelican? 应该是 make html
    4. 即可见 output 目录下 pelican.html

因为之前 github 改版，将 github pages 从 http 到 https 导致我的样式及其他页面无法正常载入
解决方法：
    设置 amayo_blog 下 pelicanconf.py 及 publishconf.py 中 文件配置 修改 
        SITEURL = 'http://zlxs23.github.io'
            为
        SITEURL = 'https://zlxs23.github.io'
    这样在 Pelican 生成页面时才能正常加载其他页面。
