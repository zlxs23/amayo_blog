Usage of Sublime Text

2016-06-06
By Amayou

Usage of Sublime Text

General

  1. ↑ ↓ ← → : 上下左右 移动光标
  2. Alt : 唤出菜单
  3. Ctrl + Shift + P : 调出 Command Palette
  4. Ctrl + ` : 调出控制台

Editing

  1. Ctrl + Enter : 在当前行下面新增一行然后调至该行
  2. Ctrl + Shift + Enter : 在当前行上面增加一行然后调至该行
  3. Ctrl + ← → : 逐词左右移动
  4. Ctrl + Shift + ← → : 逐词左右选择
  5. Ctrl + ← → : 移动当前显示区域
  6. Ctrl + Shift + ← → : 移动当前行

Selecting

  1. Ctrl + D : 选择当前光标所在词的所有出现的位置[Ctrl D 一下 则选中一个，Ctrl K 则跳过，Ctrl U则回退，Esc 则退出多重编辑]
  2. Ctrl +Shift + L : 将当前选择区域打散
  3. Ctrl + J : 将当前选中区域合并成一行
  4. Ctrl + M : 在起始括号与结尾括号进行切换
  5. Ctrl + Shift + M : 快速选中括号内内容
  6. Ctrl + Shift + J : 快速选中同级缩进的内容
  7. Ctrl + Shift + Space : 快速选中当前作用域的内容

Finding & Repalcing

  1. F3 : 跳至当前关键字下一个位置
  2. Shift + F3 : 跳至当前关键字上一个位置
  3. Alt + F3 : 选中当前关键字出现的所有位置
  4. Ctrl + F/H : 进行标准的查找/替换，之后：
    1. Alt + C : 切换大小写敏感模式(Case seneitive)
    2. Alt + W : 切换整字匹配模式(Whole matching)
    3. Alt + R : 切换正则匹配模式(Regx matching)
    4. Ctrl + Shift + H : 替换当前关键字
    5. Ctrl + Alt + Enter : 替换所有关键字匹配
  5. Ctrl + Shift + F : 多文件搜索&匹配

Jumping

  1. Ctrl + P : 跳转指定文件，输入文件名之后：
    1. @ : 符号跳转
    2. : :行号跳转

    3. : 关键字跳转

  2. Ctrl + R : 跳转到指定符号
  3. Ctrl + G : 跳转到指定行数

Window

  1. Ctrl + Shift + N : 创建一个新的窗口
  2. Ctrl + N : 在当前窗口内，创建一个新的标签
  3. Ctrl + W : 关闭当前标签
  4. Ctrl + Shift + T : 恢复刚刚关闭的标签

Screen

  1. F11 : 切换全屏
  2. Shift + F11 : 切换极简全屏
  3. Alt + Shift + 2 : 进行上下分屏
  4. Alt + Shift + 8 : 进行左右分屏
  5. Alt + Shift + 5 : 进行上下左右分屏
  6. 分屏后，使用 Ctrl + Num 调到指定屏，使用Ctrl + Shift + Num将当前屏移动到指定屏
